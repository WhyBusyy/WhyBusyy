### 💼 경력사항

#### 🏢 주식회사 루멘테라 / 프론트엔드 개발자

**기간**: 2024.05 ~ 현재

**서비스**: 약국 직거래 통합관리 플랫폼, 플랫팜

> 약국과 제약사 간의 상품 주문, 결제, 장부관리까지 한 번에 처리할 수 있는 B2B 직거래 통합관리 플랫폼으로, 약국 유통 구조의 디지털 전환을 목표로 하는 서비스입니다.

**주요 업무 및 성과**:

- ✅ SSR/CSR 혼합 렌더링 구조 전환
  - 기존 정적 페이지 생성(SSG) 방식에서 SSR/CSR 방식으로 전환
  - 서비스 기능 확장으로 인한 페이지 로딩 지연 문제 해결
  - 사용자 데이터 반영 속도 및 UX 개선
- ✅ Frontend 성능 개선 및 구조 최적화
  - Lighthouse 성능 점수 13점 → 78점 (메인페이지), 사용자 경험 개선
  - CDN 폰트를 로컬 임베디드 폰트로 전환 → 초기 렌더링 시간 단축
  - 무거운 스크립트 lazyOnLoad 적용으로 TBT 개선
  - Server Component prefetch, dynamic import 도입으로 초기 렌더링 및 전환 속도 개선
  - MUI Modal → 커스텀 모달 컴포넌트로 교체, 리렌더링 최소화
  - LocalStorage 기반 인증 로직 → NextAuth 기반 쿠키 세션 방식으로 개선, 인증 보안성과 유지보수성 강화
